## ❓ Frequently Asked Questions

**Does this software modify my existing files?**
No. The application creates separate HTML files in a folder you choose. It does not touch your other files.

**How do I save my changes?**
When you generate a slide deck, the software saves an `.html` file. You can reopen this file in your browser at any time. To change the content, return to the main application window and click "Generate" again.

**Are there hidden costs?**
The core templates in this repository are free and open for public use. Some AI agents require a personal API key from their respective providers.

**How do I remove the software?**
Open your Windows Control Panel, select "Uninstall a Program," and choose "Awesome HTML Slide Skills" from the list to remove it from your computer.

## 📖 Understanding HTML Slides

HTML slides are simply web pages designed to act like a presentation. Because they are based on web standards, they look sharp on any resolution, from small laptops to large projectors. You do not need a special viewer like PowerPoint to open them. Any standard browser renders them perfectly. This format makes it easy to share your work; you can send your slide file to a partner, and they can open it instantly without installing anything.
